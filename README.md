# ReproGen CLI - Enhanced ML Environment Generator

This branch provides an enhanced command-line interface for generating ML environments on Chameleon Cloud with better user experience, helpful prompts, and conditional logic.

---

## craft your template

### Option 1: Use the Enhanced CLI (Recommended)

```sh
# Clone this branch

git clone https://github.com/A7med7x7/ReproGen.git
cd ReproGen
git checkout cli

# Run the enhanced CLI
python reprogen.py
```

### Option 2: Use Standard Cookiecutter

```sh
cookiecutter https://github.com/A7med7x7/ReproGen.git --checkout template
```
